# タスク：ソニー製LiDAR AS-DT1 納入仕様確認のための質問リスト作成

## 背景

- 大林さんより、ソニー製LiDAR「AS-DT1」の納入仕様を確認したいとの依頼あり
- 石川さんより、AS-DT1の基本仕様等について質問事項をサマリーしておくよう指示あり
- 目的：データシートに記載のない情報や、実運用上確認が必要な事項を質問リストとしてまとめる

## やること

1. ソニー AS-DT1 の公式データシート・製品ページを調査し、公開されている基本仕様を整理する
2. データシートだけでは判断できない事項・追加確認が必要な事項を洗い出す
3. 質問リスト（サマリー）をmarkdown形式で出力する

## 調査観点

以下の観点で情報を収集し、データシートに明記されていない項目を質問候補とすること。

### 基本性能
- 測距レンジ（最小・最大）、測距精度
- 視野角（FoV）：水平・垂直
- 角度分解能
- フレームレート / スキャンレート
- 検出対象の最小反射率条件

### 出力・インターフェース
- 出力データフォーマット（点群形式など）
- 通信インターフェース（Ethernet / USB / その他）
- 対応プロトコル

### ソフトウェア・開発環境
- SDK提供の有無と対応OS
- ROS / ROS2 対応状況
- サンプルコード・ドキュメントの充実度
- ファームウェアアップデート方法

### 環境・耐久性
- 動作温度範囲
- 防塵・防水等級（IP等級）
- 耐振動・耐衝撃性能
- 屋外使用時の太陽光干渉耐性

### 電源・物理仕様
- 入力電圧・消費電力
- 外形寸法・重量
- 取付方法・マウント仕様

### 納入・商務条件
- 納期（リードタイム）
- 最小発注数量（MOQ）
- 価格帯（参考価格）
- 保証期間・サポート体制
- 評価機貸出の可否

## 出力形式

以下の構成でmarkdownファイルを出力すること：

```
# ソニー製LiDAR AS-DT1 仕様確認・質問リスト

## 1. データシートから判明している仕様一覧
（表形式で整理）

## 2. データシートに記載がない／不明確な事項（質問リスト）
（番号付きリストで、質問の意図も簡潔に添える）

## 3. 補足・備考
（調査中に気づいた点など）
```
