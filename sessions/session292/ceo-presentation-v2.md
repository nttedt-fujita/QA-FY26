# 4. QRコードロット管理の概要

## 採番ルール

```
TA - 20260325 - 001
│      │        │
│      │        └─ 連番（3桁）
│      └────────── 入荷日
└─────────────── 部品コード
```

## 運用フロー

```
入荷 → QR発行 → 検査 → 梱包変更 → 入庫 → 出庫
        ↓        ↓        ↓        ↓      ↓
      スキャン  スキャン  スキャン  スキャン スキャン
        ↓        ↓        ↓        ↓      ↓
      （全て記録に残る → 隠れコストも可視化）
```
