## 4. PSIの構成要素（確度別）

```
PSI Excel
├── 需要側（Agriシートの情報由来）
│   ├── 受注数（出荷ベース）     ← Agri集計 → PSI [A]
│   │                             kintone → Agri [B]（転記方法は未確認 [D]）
│   ├── コメント（顧客内訳）     ← Agriの顧客名と一致 [A]、手動転記 [B]
│   └── 商流別受注数詳細シート   ← Agriからの集計と推定 [D]
│
├── 供給側（ソース不明）
│   ├── 入庫 / 製造              ← ソース不明 [D]（「ナカヨ」の記載あるがGEN由来かは未確認）
│   ├── 出庫実績                 ← ソース不明 [D]（群馬通商出庫数との直接一致なし）
│   ├── 在庫(NTT) / 在庫(SL)    ← ソース不明 [D]（コメントに「群馬倉庫」記載 [A]）
│   └── 実在庫                   ← ソース不明 [D]
│
├── 予測（人手由来）
│   ├── 出荷予測                 ← 長屋さん [C — S313ヒアリングのみ]
│   └── 出荷予測（経営ベース）   ← 滝澤さん [A — PSI凡例で確認]
│
└── 自動計算（Excel数式）         ← [A — 凡例「青塗りつぶし = 自動計算」]
    ├── 在庫（受注ベース残）     ← = 前月在庫 + 入庫 - 受注
    ├── 在庫（納品ベース残）     ← = 前月在庫 + 入庫 - 納品
    └── 在庫（出荷予測ベース残） ← = 前月在庫 + 入庫 - 予測
```
