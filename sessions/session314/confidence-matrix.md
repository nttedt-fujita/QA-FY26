# 情報の確度整理

**作成**: Session 314 (2026-03-26)
**目的**: 分析で得た情報を確度別に整理し、推測と事実を明確に分離する

---

## 確度レベルの定義

| レベル | 意味 | 根拠の基準 |
|--------|------|-----------|
| **A: 分析で確認済み** | データの突合で数値的に確認した事実 | 複数月のデータが一致 |
| **B: 複数情報源から推定** | ヒアリング＋分析結果が符合する推定 | 聞いた話とデータが矛盾しない |
| **C: 単一情報源からの推定** | 1つの情報源のみに基づく推定 | 他の裏付けなし |
| **D: 推測** | データ的裏付けなし | 構造から「こうではないか」と考えた仮説 |
