# 受入検査工数 月別分析方針

**決定日**: 2026-03-06 (Session 19)

## 分析フレームワーク: パレート分析 + 4M分解

### パレート分析（ABC分析）

月別に「上位20%の部品が全体工数の何%を占めるか」を算出する。
80/20の法則に基づき、重点管理すべき部品を特定する。

### 4M分解

| 軸 | 切り口 | 分析内容 |
|---|--------|---------|
| Man（作業者） | 作業者ごと | 件数・工数・平均工数のばらつき |
| Material（部品） | 品名・カテゴリごと | 工数割合、パレート順位の変動 |
| Method（検査方法） | 検査内容ごと | 外観/寸法/機能別の工数差 |
| Machine（設備・場所） | 検査場所・治具ごと | 使用傾向（データがあれば） |

---

## 出力ファイル構成

### サマリーCSV（1枚）

`summary/月別サマリー.csv`

各月の行に以下を含む:
- 総件数、総工数(h)、部品種類数
- パレート指標（上位20%品名が占める工数割合）
- カテゴリ別割合（メカ/エレキ/Api）
- 作業者数、最多担当者
- 工数トップ3部品

### 詳細CSV（切り口別）

| ファイル | 内容 |
|---------|------|
| `detail/月別×部品.csv` | 各月の品名ごとの件数・工数（パレート分析用） |
| `detail/月別×作業者.csv` | 各月の作業者ごとの件数・工数・平均（Man） |
| `detail/月別×カテゴリ.csv` | メカ/エレキ/Api別の月別件数・工数（Material） |
| `detail/月別×検査内容.csv` | 外観/寸法/機能別の月別件数・工数（Method） |

## 前提・注意事項

- **表記揺れ未解消**: 品名・作業者名に表記揺れがある。名寄せは別途検討
