### 3. RTK FIX時間について

**質問**:

### 4. TTFF（Time To First Fix）について

**業界標準**:
| 起動モード | 典型値 |
|-----------|--------|
| Cold Start | 2-4分 |
| Warm Start | ≤45秒 |
| Hot Start | ≤22秒 |

**質問**:
- [ ] TTFFは現在確認していますか？
