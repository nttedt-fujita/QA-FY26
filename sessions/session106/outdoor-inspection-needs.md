## 3. NAV-SAT vs NAV-SIG の選択

**結論**: **NAV-SIG**を使用

| 観点 | NAV-SAT | NAV-SIG | 選択理由 |
|------|---------|---------|---------|
| 粒度 | 衛星単位 | **信号単位（L1/L2別）** | L1/L2別が必要 |
| C/N0 | 衛星代表値 | **信号ごとの値** | L1/L2別の比較が必要 |
| 仰角/方位角 | **あり** | なし | NAV-SATから取得 |
| 用途 | スカイプロット | **受信感度評価** | 主目的は受信感度 |

**実装方針**:
- 受信感度評価: **NAV-SIG**（L1/L2別C/N0）
- スカイプロット: NAV-SAT（仰角・方位角）
- または: NAV-SIG + NAV-SATを組み合わせ

## 4. MON-SPAN仕様まとめ

[ubx-spec-memo.md](ubx-spec-memo.md)より。

| 項目 | 値 |
|------|-----|
| Class/ID | 0x0a 0x31 |
| Type | Periodic/polled |
| スペクトラムデータ | **256点** (U1[256]) |
| 単位 | dB |
| 周波数計算 | f(i) = center + span × (i - 127) / 256 |

**128MHz/256点の場合**:
- span = 128,000,000 Hz
- res = 500,000 Hz (500kHz)

**注意**: 比較分析用であり、絶対値の精度は高くない。
