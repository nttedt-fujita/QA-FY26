# 3. 現場調査での発見

## 2026年3月 田原さんヒアリング実施

<br>

**想定**していた作業フロー:
```
入荷 → 受入検査（員数確認）→ 入庫
```

**実態**:
```
入荷 → 開封 → 検査 → 梱包変更作業 → 入庫
                           ↑
                       隠れコスト
```

# 3. 問題の構造

## 測定バイアス

「測定しやすいもの」だけを測定していた

| 可視化 | コスト |
|--------|--------|
| ✅ 受入検査 | 記録あり |
| ❌ 梱包変更作業 | 記録なし（隠れコスト） |
| ❌ 小分け再作業 | 記録なし |

**→ 全体のコストが把握できていない**

# 4. 今年の目標

## 可視化に注力する

**理由**: まだ何も数字になっていない

### Phase 1 の目標（3ヶ月）

- 作業フロー全体の可視化（SIPOC完了済み ✅）
- 暗黙知の外部化（進行中）
- 隠れコストの特定と定量化（進行中）

# 5. 根本解決策

## 梱包仕様（姿見）の標準化

### 現状の問題

- 納品時の梱包 → 組立業者への出庫に適していない
- うちで梱包変更作業が発生

### 解決策

- **納品時の梱包仕様を、出庫に適した形に統一**
- うちでの作業を省略
- 業者との調整に時間をかけて実施
