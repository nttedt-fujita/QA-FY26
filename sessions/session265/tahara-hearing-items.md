**日付**: 2026-03-19 (Session 265)
**目的**: 受入検査作業のプロセスが実態と合っているか確認し、宇枝さんへの報告資料に反映

## 前提: 工数の可視化ができていない状態

**現状の課題**:
- 作業工数が可視化されていない（田原さんに聞いても正確な数字が出せない）
- この「分からない」状態自体が問題
- 工数記録の仕組みが存在しない

**今回の確認目的**:
- まず「何の作業をしているか」（プロセス）を正確に把握する
- 工数の数字は後回し（仕組みを作ってから測定）

---

## 確認項目

### 1. プロセスの正確性確認

SIPOC資料の「Process」列を見ながら確認：

| # | プロセス（SIPOC資料） | 実態と合っているか？ | 修正・追加すべき点 |
|---|---------------------|---------------------|------------------|
| 1 | 部品入荷 | ✅ / ❌ / 部分的 |  |
| 2 | 検査準備（梱包確認） | ✅ / ❌ / 部分的 |  |
| 3 | 検査実施（外観・員数） | ✅ / ❌ / 部分的 |  |
| 4 | 記録作成（Excel入力） | ✅ / ❌ / 部分的 |  |
| 5 | 入庫/不合格処理 | ✅ / ❌ / 部分的 |  |

**質問**:
- 「このプロセスは実際の作業と合っていますか？」
- 「抜けている作業はありませんか？」
- 「順序は正しいですか？」

---

### 2. 各プロセスの詳細（サブステップ）確認

特に「検査準備（梱包確認）」の詳細を確認：

**質問**:
- 「検査準備」では具体的にどんな作業をしていますか？
  - 例: 開梱 → 梱包状態確認 → **梱包変更作業** → 検査準備完了

**Session 260で判明したこと**:
- 梱包変更作業が存在（隠れコスト）
- この作業が受入検査工数の大半を占めている可能性

**確認すべきこと**:
- 梱包変更作業はどのプロセスに含まれるか？（プロセス2？別プロセス？）
- 梱包変更作業は毎回発生するか？一部のサプライヤーのみか？
- 他にも「隠れた作業」はないか？

---

### 3. 抜けている作業・余計な作業の確認

**質問**:
- SIPOC資料に載っていないが、実際にはやっている作業はありますか？
  - 例: サプライヤーへの問い合わせ、検査基準書の確認、測定器具の準備など
- 逆に、SIPOC資料に載っているが実際にはやっていない作業はありますか？

---

### 4. 工数記録の実態（参考情報）

**質問**（参考として確認）:
- 現在、作業時間は記録していますか？
  - Excel「検査工数(分)」列はどうやって記入していますか？
  - ストップウォッチで測定？ 記憶で記入？ 概算？

**補足**:
- 正確な数字は不要（今は測定できない状態であることを確認するだけ）
- 「分からない」「記録していない」という回答も重要な情報
