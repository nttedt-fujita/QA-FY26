## 1. 品質コスト（CoQ）とは

**品質コスト（Cost of Quality; CoQ）** = 顧客要求を満たす品質を確保・維持するために発生する全てのコスト。

> ポイント: 「品質を高めるためのコスト」だけでなく、「品質不良によって発生するコスト」も含む。

---

## 2. PAF法による4分類

**PAF法（Prevention-Appraisal-Failure Approach）** = 品質コストを4つに分類するフレームワーク。ISO 9000でも採用。

```
品質コスト（CoQ）
├── 適合コスト（Conformance Cost） ← 品質を"作り込む"ためのコスト
│   ├── P: 予防コスト（Prevention Cost）
│   └── A: 評価コスト（Appraisal Cost）
└── 不適合コスト（Non-Conformance Cost） ← 品質不良による損失
    ├── F（内部）: 内部失敗コスト（Internal Failure Cost）
    └── F（外部）: 外部失敗コスト（External Failure Cost）
```

### 各コストの定義と具体例

#### P: 予防コスト

**定義**: 品質問題が発生しないようにするための活動コスト

| 具体例 | 自社への適用例 |
|--------|---------------|
| 品質計画・設計レビュー | 受入検査基準書の策定 |
| 教育・訓練費 | 検査員トレーニング |
| サプライヤ品質指導費 | ナカヨ・渡辺への品質指導 |
| 工程改善活動 | 組立工程の標準化 |
| 品質協定書締結 | サプライヤとの品質協定書 |

#### A: 評価コスト

**定義**: 品質要件を満たしているか確認・測定するためのコスト

| 具体例 | 自社への適用例 |
|--------|---------------|
| 受入検査費（IQC） | 部品入荷時の検査工数 |
| 工程内検査費（PQC） | 組立中間工程の検査 |
| 最終製品検査費（FQC） | 出荷前の全数/抜取検査 |
| 試験・測定設備の維持費 | 測定機器の校正費用 |

#### F（内部）: 内部失敗コスト

**定義**: 製品が顧客に届く前に発見された品質不良のコスト

| 具体例 | 自社への適用例 |
|--------|---------------|
| 手直し・再加工費 | 不良品の修正作業工数 |
| スクラップ・廃棄費 | 仕損品の廃棄コスト |
| 再検査費 | 修正後の再検査工数 |
| 工程ダウンタイム | 不良対応による生産停止時間 |

> **仕損品（自社の事例）**: S324bで確認 → kintone出庫データで仕損24%（78件中19件）

#### F（外部）: 外部失敗コスト

**定義**: 製品が顧客に届いた後に発見された品質不良のコスト（最も高コスト）

| 具体例 | 自社への適用例 |
|--------|---------------|
| クレーム対応費 | 顧客からの不良品クレーム対応 |
| 返品・無償交換費 | 不良品の回収・再送料 |
| リコール費用 | 市場回収・改修費用 |
| 賠償・訴訟費用 | 品質事故による損害賠償 |
| ブランド毀損 | 顧客信頼の喪失（定量化困難） |

## 3. CoQの業界目安と理想的な構成

### 品質コスト総額の目安（CoQ率）

```
CoQ率 = 品質コスト合計 ÷ 売上高 × 100
```

| CoQ率 | 評価 |
|-------|------|
| 〜5% | 優秀（品質管理が成熟） |
| 5〜10% | 標準的（改善余地あり） |
| 10%以上 | 要改善（失敗コストが過大） |

> **参考**: 品質管理が成熟していない企業では売上高の15〜25%が品質コストとなっているケースも。

### 理想的なコスト構成の方向性

```
【現状（品質管理が未成熟な状態）】
予防コスト: 小 ← ここが少ない
評価コスト: 中
内部失敗コスト: 大
外部失敗コスト: 大 ← ここが大きい（最悪）

【目指す状態（成熟した品質管理）】
予防コスト: 中 ← 増やす（投資）
評価コスト: 中
内部失敗コスト: 小 ← 減る（効果）
外部失敗コスト: 極小 ← 減る（最大効果）
```

### 予防コスト投資の費用対効果

> **経験則**: 予防コスト1円の投資 → 失敗コスト3〜10円の削減

理由: 問題が後工程で発覚するほどコストは指数的に増加する（1:10:100の法則）。

```
設計段階での修正: 1
製造段階での修正: 10
顧客納品後の修正: 100
```

## 4. 品質コストレポートの作成方法

### ステップ

1. **コスト項目の定義**: 自社の業務フローに合わせてPAF分類を確定
2. **データ収集**: 各部門から工数・費用データを収集
3. **集計・可視化**: カテゴリ別の金額と売上高比率を算出
4. **傾向分析**: 月次・四半期でトレンドを確認
5. **改善アクション**: 最も大きいコスト項目から対策

### 品質コストレポートの構成例

| カテゴリ | 金額（万円/月） | 売上比率 | 前月比 |
|---------|----------------|---------|--------|
| 予防コスト | XX | X.X% | ±X% |
| 評価コスト | XX | X.X% | ±X% |
| 内部失敗コスト | XX | X.X% | ±X% |
| 外部失敗コスト | XX | X.X% | ±X% |
| **合計（CoQ）** | **XX** | **X.X%** | ±X% |

## 5. 自社への適用（DX効果分析の観点）

### 現状の課題と品質コストの対応

| 課題 | PAF分類 | 確認セッション |
|------|---------|--------------|
| 仕損品24%（78件中19件） | 内部失敗コスト | S324b |
| 仕損品追跡の工数（手動管理） | 内部失敗コスト | S319-327 |
| 受入検査工数（プロポが最大） | 評価コスト | S25ヒアリング |
| 品質協定書なし（ナカヨ/渡辺） | 予防コストの欠如 | S25 |
| 群馬通商Excel手動フラグ管理 | 評価コスト（転記工数） | S327 |

### DX後のCoQ改善仮説

```
【現状】
仕損品管理: Excel手動 → 追跡困難 → 外部失敗コスト（顧客クレーム）リスク

【DX後（DB化・自動化）】
仕損品追跡: DB自動化 → 追跡可能 → 内部失敗コスト削減、外部失敗コスト削減
kintone→GEN自動化: タイムラグ解消 → 在庫精度向上 → 内部失敗コスト削減（廃棄・過剰在庫）
```

**定量化の方向性**（QA-24で詳細算出）:
- 仕損品19件/78件の追跡工数: X時間/件 × 19件/月 = Y時間/月
- 再送・廃棄コスト: Z円/件 × 19件/月 = W円/月

## 6. CoQとIQC/8Dの連携

Session 25で整理したQCフレームワークとの関係:

```
IQC（受入検査）
└── 評価コスト: 検査工数
└── 内部失敗コスト: 不良品の処理コスト
    ↓（見逃した場合）
外部失敗コスト: 顧客クレーム・返品

8Dレポート（問題解決フレームワーク）
└── D4（根本原因）→ 予防コスト投資の根拠
└── D6（効果確認）→ CoQ改善効果の測定
```

> CoQは「どこに問題があるか」を金額で可視化するツール。
> 改善活動（8D）の前後でCoQを比較することで効果を定量化できる。
