# Session 44 計画

**目的**: カウンター画面（フロントエンド）実装

---

## メインタスク

1. **カウンター画面実装**（実装計画より）
   - 検査開始フォーム（ロット/検査項目/作業者選択）
   - カウンター表示（合格/不合格/不問）
   - 操作ボタン（取り消し、メモ追加、検査終了）
   - 検査終了ダイアログ

---

## 追加タスク（Session 43からの引き継ぎ）

2. **テスト改善**
   - 複数取り消しテスト: LIFO順序を検証するケース追加
   - 「取り消してもカウントが残る」パターン追加
   - 例: `pass→pass→fail→取消` → 合格=2, 不合格=0

3. **ADR作成**: 型明示ルール
   - 本番コードでは `var x int = 10` のように型を明示する
   - プロトタイプでは `:=` を許容

4. **TDDスキル更新**
   - Phase 2成果物（テストシナリオ）をドキュメント化する推奨を追加
