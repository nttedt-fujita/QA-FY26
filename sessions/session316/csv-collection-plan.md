# kintone CSV収集計画

**目的**: 全体業務フロー・担当者・作業の流れを把握するdrawio図を作るための構造把握
**作業環境**: Windows PC（kintoneアクセス）

---

## CSV出力の仕組み（マニュアルより）

- 一覧画面から「…」→「ファイルに書き出す」
- **出力列は選択可能**（左側パネルから移動して選ぶ）
- **一覧のフィルタ条件がそのままCSV出力に反映される**
- → つまり、一覧ビューによって出力内容が変わる

---

## スクショ取得リスト

各アプリで以下を撮る:

| # | 撮るもの | 目的 |
|---|----------|------|
| A | **一覧画面**（デフォルトビュー） | フィールド構成の把握 |
| B | **一覧ビュー切替メニュー**（左上クリック） | どんなビューがあるか把握 |
| C | **CSV出力の列選択画面**（「…」→書き出す→列選択） | 全フィールド一覧の把握 |
| D | **レコード詳細画面**（1件クリック） | セクション構成・項目の把握 |

※CのCSV出力列選択画面が**一番重要**。全フィールドが一覧で見える。

## 推奨作業手順

### ステップ1: スクショ（全アプリ共通）

1. toppage3の「さらに表示」→ **全アプリ一覧のスクショ**（番号体系全体の把握）
2. P0の5アプリについて、それぞれA〜Dのスクショを撮る
3. 特にCの**CSV出力列選択画面**を必ず撮る

### ステップ2: CSV出力（P0のみ先行）

1. 各アプリのCSV出力画面で**全列を選択**して出力
2. データ件数が多い場合は直近1年分など絞る
3. CSVファイルはsessions/session316/csv/に保存

### ステップ3: 構造分析

1. CSVのヘッダー行からフィールド一覧を抽出
2. アプリ間の紐づき（外部キー的なもの）を特定
3. ER図的な構造をdrawioで作成

### ステップ4: 全体フロー図作成

1. マニュアルのフロー情報 + CSVの実データ構造を統合
2. drawioで全体業務フロー図を作成

---

## 注意事項

- CSV出力時の一覧ビューでフィルタが効く → **「すべて」の一覧を選んでから出力**すること
- 件数が多いアプリは全件出力に時間がかかる可能性あり
- 個人情報を含むアプリ（取引先担当者等）はCSV出力の取り扱い注意
