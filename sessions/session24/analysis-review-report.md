**観察**:
- **2025-12を境に主担当が西村→杉山へ移行**
- 杉山の平均工数/件が高い（複雑案件が多い）
- 複数人での検査（西村・西浦など）が一定数存在

### Machine（検査治具）

データからは治具情報が直接取得できないが、検査内容から推測:
- **外観検査**: 目視+ルーペ（標準的）
- **寸法検査**: ノギス、マイクロメーター
- **通電検査**: テスター、専用治具
- **キャリブレーション**: FC専用治具
- **ログ抜き**: PC接続ツール

## 4. 発見事項と改善提案

### 発見事項

1. **工数の二極化**
   - 外観のみ: 20〜40分/件（標準的）
   - 特殊検査: 4〜24時間/件（PCM、チャージャ等）

2. **担当者移行**
   - 西村→杉山への引継ぎが2025年末に発生
   - 引継ぎ後の工数/件は上昇傾向（習熟曲線）

3. **検査内容の表記揺れ**
   - 名寄せで約40%統合したが、まだ残存
   - 「外観+員数」と「外観、員数」など

4. **高工数案件の集中**
   - 特定月に高工数案件が集中すると、その月の総工数が跳ね上がる
   - 計画的な分散が必要

### 改善提案

| 優先度 | 提案 | 期待効果 |
|--------|------|----------|
| **P0** | 高工数品目（PCM、チャージャ等）の事前スケジューリング | 月次工数の平準化 |
| **P1** | 検査内容の標準化（入力フォーム改善） | データ品質向上 |
| **P1** | 複数人検査のルール明確化 | 工数記録の精度向上 |
| **P2** | 新担当者への習熟支援 | 工数/件の安定化 |
