## 1. AI化の判断基準

AI検査を導入する部品の選定基準:

| 基準 | 説明 | 理由 |
|------|------|------|
| **価格が高い部品** | 不良見逃しの損失が大きい | ROI観点で効果大 |
| **影響が大きい部品** | 機能・安全に関わる | 品質リスク低減 |
| **まとめて効率化できる** | 同じ撮影環境で複数部品検査可能 | 投資効率 |
| **写真撮影コスト < AI価値** | 撮影の手間をかける価値がある | 運用コスト考慮 |

**重要**: 写真撮影自体が人的コストになるため、AIに任せる価値がある部品を選定する必要がある。

## 2. 外観検査の候補（仮）

### 2.1 プロポ

| 項目 | 内容 |
|------|------|
| **検出したい不良** | シルク印刷のカケ、外装部打痕 |
| **現行検査** | 外装／電装品動作チェック／シール貼り |
| **過去不良実績** | アンテナ不具合1個、外装部打痕12個（60個中） |
| **AI化の理由** | 価格高い、影響大きい |

**シルク印刷カケとは**: プロポ本体に印字されている文字・ロゴがカケている状態

### 2.2 ペラ（FA28.2*9.2 L/R）

| 項目 | 内容 |
|------|------|
| **検出したい不良** | キズ、汚れ、曇りキズ、補修跡 |
| **現行検査工数** | 120-200分/100個（複数人） |
| **備考** | 毎回不具合箇所が異なるため、都度判断依頼 |

### 2.3 クラック検出（難易度高）

- 要求としてはあるが、技術的ハードルが高い
- 学習データの準備が困難
- Phase 2以降の検討項目
