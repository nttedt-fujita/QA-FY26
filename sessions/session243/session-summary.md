## 実施内容

1. **員数確認・外観検査の対象整理**
   - ネジは対象外（小さすぎて写真判別困難）
   - ケースに入っているもの、写真で判別しやすいものが候補
   - 外観検査: シルク印刷のカケ、クラック（難しい）

2. **AI化の判断基準を整理**
   - 価格が高い部品
   - 影響が大きい部品
   - まとめて効率化できるもの
   - 写真撮影コスト < AI価値

3. **受入検査CSVデータの確認**
   - Session 235のCSVを参照
   - 部品ごとの検査工数を確認
   - プロポ: 80分/60個、外装部打痕12個の不良実績

4. **QAフレームワークとの整合性確認**
   - Session 25の品質管理フレームワーク調査を参照
   - IQCでの位置づけを明確化
   - トレーサビリティは必要（ロット＝入荷タイミング）
   - AQLは要相談（部品・文脈依存）
