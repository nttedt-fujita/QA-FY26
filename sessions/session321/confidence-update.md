# 確度更新（Session 321: GEN画面直接確認による）

**目的**: S317ドメインモデルの確度をGEN画面の直接確認結果で更新する
**参照**: [session317/domain-model-as-is.md](../session317/domain-model-as-is.md)
**分析手法**: GEN画面スクショ + テキストコピー + CSVエクスポート

---

## 確度レベルの定義（S314準拠、再掲）

| レベル | 意味 | 根拠の基準 |
|--------|------|-----------|
| **A: 確認済み** | データ分析・CSV等で直接確認した事実 | 複数ソースが一致、またはデータで確認 |
| **A(M): マニュアル記載** | マニュアルに書いてある事実 | 現在の実態と異なる可能性あり |
| **A(G): GEN画面確認** | GEN画面で直接確認した事実 | **S321で新設**。GENにログインしてデータの存在を目視確認 |
| **B: 複数情報源から推定** | ヒアリング＋分析結果が符合する推定 | 聞いた話とデータが矛盾しない |
| **C: 単一情報源からの推定** | 1つの情報源のみに基づく推定 | 他の裏付けなし |
| **D: 推測** | データ的裏付けなし | 構造から「こうではないか」と考えた仮説 |
