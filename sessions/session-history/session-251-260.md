## Session 256 (2026-03-18)

**概要**: AI検査コスト計算の変数確認・再計算・方向転換

**実施内容**:
1. 計算変数の出典付き整理（堂々巡り防止）
2. 撮影回数10回/台での再計算
3. 複数シナリオでのコスト比較
4. 小笠原さんからの新情報による方向転換

**確定した変数**:

| 変数 | 値 | 出典 |
|------|-----|------|
| 目視検査時間 | 3分/個 | ユーザー確認（Session 253頃） |
| 年間製造台数 | 500台/年 | ユーザー確認（Session 256） |
| プロポ撮影回数 | 2回/台 | ユーザー確認（Session 256） |
| ペラ撮影回数 | 8回/台 | ユーザー確認（Session 256） |

**計算結果**:
- 10回/台・20秒/回の場合: 純削減2.4万円/年、回収66〜178年
- 6回/台に減らしても: 純削減7.5万円/年、回収21〜58年
- **結論**: プロポ・ペラの外観検査AI化は500台/年規模では回収困難

**方向転換**（小笠原さんからの新情報）:
- 旧アプローチ（外観検査）は旨味少ないと認識済みだった
- 新アプローチ: **員数確認**（ガスケット、グロメット等の小物部品をAIでカウント）
- バリなどの不具合検出も検討対象

**次セッション**: [session257/session-plan.md](../session257/session-plan.md) — 対象部品のリストアップ（Excel全体から）、田原さんヒアリング準備
