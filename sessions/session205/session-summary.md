## 実施内容

### 複数台検査フローの要件確認

ユーザーとの対話で以下を決定:

| 項目 | 決定 |
|------|------|
| 検査方式 | 一括・個別の両方対応 |
| DB登録タイミング | 検査完了時に自動登録（現行踏襲） |
| DB登録順 | USBポート順（ttyUSB0 → 1 → 2） |
| 検査後フロー | 結果表示 → 点滅で識別 → 取り外し |

### 確定したフロー

```
1. 複数台接続
2. 画面に一覧表示
3. 「全台検査」or「この1台を検査」
4. 検査実行
5. 検査完了 → 自動でDB登録（USBポート順）
6. 結果表示（OK/NG）
7. 点滅ボタンで識別 → シール貼り/対処 → 取り外し
8. 次のロットへ
```
