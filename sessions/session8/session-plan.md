# Session 8 計画

**前セッション**: Session 7 (2026-03-04)

## 目的

Session 7で途中になったEARS要求仮説の整理を完了し、ヒアリング計画を具体化する

## やること

### 1. 仮説 → 誰に聞くかマッピング（Session 7の続き）

- 各仮説を「確認済み」vs「推測（要ヒアリング）」に最終分離
- 推測の仮説ごとに、誰に聞けば検証できるかをマッピング
- 対象者: 石川さん / 小板橋さん / 現場担当者

### 2. ヒアリング質問リストの更新

- Session 5の21問リストをEARS仮説に基づいて再構成
- 各質問にEARS要求IDを紐付け（何のために聞くかを明確に）
- 質問の優先順位付け

### 3. Excelデータの詳細分析・図表化

- 不良傾向の可視化（サプライヤ別、品目別、時系列）
- 備考列から抽出した課題の分類・整理
- 「不問」「要協議」「基準書未整備」の件数整理

### 4. （時間があれば）最小構成モックアップの検討

- 分析結果とヒアリング内容がまとまったら、最小構成のモックアップ方針を議論
