## 主な発見・決定

| 項目 | 内容 |
|------|------|
| アプローチ | **Webサーバー型**（ブラウザでアクセス、触る人がやりやすい） |
| パフォーマンス最重視 | **Rust + Actix-web**（19-20k req/s） |
| UBXパース信頼性 | **自前実装**（仕様書ベース、必要な5-6メッセージのみ）or **ubxlib（公式）** |
| 衛星位置計算 | **F9PのNAV-SATから直接取得**（計算ライブラリ信頼性問題を回避） |

**推奨案**: **案A（Rust + Actix-web + 自前UBXパース）**
