## 1. QA（品質保証）とQC（品質管理）の違い

| 項目 | QA（Quality Assurance） | QC（Quality Control） |
|------|------------------------|----------------------|
| 定義 | 品質を**保証するための活動** | 品質を**確認する活動** |
| 視点 | **ユーザー視点** | **現場の作り手視点** |
| 範囲 | 全ライフサイクル | 製造工程が中心 |
| 時間軸 | 長期的・継続的 | 工程ごと・短期的 |

> 出典: [ISOプロ — QAとQCの違い](https://activation-service.jp/iso/column/7762)

### ISO 9000での体系

```
品質マネジメント（QM）
  ├── 品質管理（QC）
  ├── 品質保証（QA）
  └── 品質改善（QI）
```

## 2. ISO 9001:2015 品質マネジメント7原則

| # | 原則 | 内容 |
|---|------|------|
| 1 | 顧客重視 | 顧客のニーズを理解し、期待を超える |
| 2 | リーダーシップ | トップがQMSに責任を持つ |
| 3 | 人々の積極的参加 | 全員が役割を理解し参加する |
| 4 | プロセスアプローチ | 業務をプロセス単位で管理・改善 |
| 5 | 改善 | 常に改善方法を探求 |
| 6 | **客観的事実に基づく意思決定** | データと情報に基づいて判断 |
| 7 | 関係性管理 | ステークホルダーと対等な関係を維持 |

**藤田さんへのポイント**: 原則6が4ミッション全てに通底する考え方。

## 3. 受入検査（AQL抜取検査）

**規格**: JIS Z 9015-1 (= ISO 2859-1)

**手順**:
1. ロットサイズ・AQL・検査水準を確認
2. サンプル文字を特定
3. サンプルサイズと合格判定個数を決定

**具体例**: ロット500個、検査水準II、AQL 2.5% → サンプル50個、3個以下で合格

## 4. 工程不良管理の基本ツール

### QC7つ道具

チェックシート、パレート図、特性要因図（魚骨図）、ヒストグラム、散布図、管理図、グラフ

### 8D問題解決手法

D0(準備) → D1(チーム編成) → D2(問題定義) → D3(暫定対策) → D4(根本原因分析) → D5(恒久対策選定) → D6(実施) → D7(再発防止) → D8(チーム評価)

### SPC（統計的工程管理）

管理図でUCL/LCLを設定し、工程が統計的管理状態にあるかを監視する。
