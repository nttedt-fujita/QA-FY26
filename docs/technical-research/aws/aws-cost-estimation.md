## 前提条件

| 項目 | 値 |
|------|-----|
| リージョン | 東京 (ap-northeast-1) |
| ユーザー数 | 5人程度（品質Gチーム） |
| データ量 | 小規模（1日数件〜数十件の検査記録） |
| 画像保存 | 検査写真・不良品写真（1枚あたり1-5MB想定） |
| 可用性 | 業務時間内に使えればOK（24/365は不要） |
| セキュリティ | 社内データ、外部公開なし |

## 構成パターン

### パターンA: サーバレス構成（推奨候補）

```
[ブラウザ/タブレット]
    │
    ├── CloudFront → S3 (フロントエンド配信)
    │
    └── API Gateway → Lambda → RDS PostgreSQL
                                    │
                        S3 (検査画像保存)
                                    │
                        Cognito (認証)
```

| サービス | スペック | 月額概算 |
|---------|---------|---------|
| RDS PostgreSQL | db.t4g.micro (2vCPU, 1GB RAM), Single-AZ, 20GB SSD | ~$18-25 |
| Lambda | 月数千リクエスト想定（無料枠内） | ~$0 |
| API Gateway | 月数千リクエスト想定 | ~$0-1 |
| S3 | フロントエンド + 画像保存 (数GB) | ~$0-1 |
| CloudFront | フロントエンド配信 (微量) | ~$0-1 |
| Cognito | 5ユーザー（無料枠: 50,000 MAU） | $0 |
| **合計** | | **~$20-30/月 (約3,000-4,500円)** |

**メリット**:
- Lambdaは使った分だけ課金、アイドル時ゼロ
- RDS以外はほぼ無料枠内で収まる
- スケールの心配が不要

**デメリット**:
- RDSは停止しても完全無料にはならない（ストレージ課金）
- Lambda + API Gatewayの開発・デバッグが若干面倒
- コールドスタートの遅延（数百ms〜数秒）

---

### パターンB: Lightsail（VPS型・最シンプル）

```
[ブラウザ/タブレット]
    │
    └── Lightsail インスタンス
         ├── Nginx (リバースプロキシ)
         ├── FastAPI / Django (バックエンド)
         ├── PostgreSQL (同居 or Lightsail DB)
         └── 画像ファイル (ローカル or S3)
```

| サービス | スペック | 月額概算 |
|---------|---------|---------|
| Lightsail インスタンス | 1Core, 1GB RAM, 40GB SSD, 2TB転送 | $5 |
| Lightsail DB（オプション） | Micro, 1vCPU, 1GB RAM, 40GB | $15 |
| S3（画像のみ） | 数GB | ~$0-1 |
| **合計（DB同居）** | | **~$5-6/月 (約750-900円)** |
| **合計（DB分離）** | | **~$20-21/月 (約3,000-3,200円)** |

**メリット**:
- 月額固定で予測しやすい
- 1台にすべて載せるので構成がシンプル
- SSHで直接操作でき、開発しやすい

**デメリット**:
- サーバ管理（OS更新、セキュリティパッチ）が自分の仕事
- スケールが必要になったら構成変更が必要
- DB同居の場合、バックアップは自分で設定

---

### パターンC: Amplify + RDS

```
[ブラウザ/タブレット]
    │
    └── Amplify Hosting (React/Next.js)
         │
         └── Amplify Backend (AppSync or REST)
              │
              └── RDS PostgreSQL
```

| サービス | スペック | 月額概算 |
|---------|---------|---------|
| Amplify Hosting | ビルド + ホスティング（小規模） | ~$1-5 |
| Amplify Backend | API処理 | ~$0-2 |
| RDS PostgreSQL | db.t4g.micro, Single-AZ | ~$18-25 |
| **合計** | | **~$20-32/月 (約3,000-4,800円)** |

**メリット**:
- フルスタック開発の統合環境
- CI/CDが組み込み
- フロントエンド開発が楽

**デメリット**:
- Amplifyの学習コスト
- AWS依存が強くなる
- バックエンドの柔軟性がやや低い

## コスト比較サマリー

| パターン | 月額 | 年額 | 保守負荷 | 開発しやすさ |
|---------|------|------|---------|------------|
| **A: サーバレス** | ~$25 (~3,750円) | ~$300 (~45,000円) | 低 | 中 |
| **B: Lightsail (DB同居)** | ~$5 (~750円) | ~$60 (~9,000円) | 中〜高 | 高 |
| **B: Lightsail (DB分離)** | ~$20 (~3,000円) | ~$240 (~36,000円) | 中 | 高 |
| **C: Amplify + RDS** | ~$25 (~3,750円) | ~$300 (~45,000円) | 低 | 中 |

※ 1ドル = 150円で換算

## 無料枠について

AWS無料枠（12ヶ月間）を活用すると、初年度はさらに安くなる:

| サービス | 無料枠 |
|---------|--------|
| RDS | db.t4g.micro 750時間/月（12ヶ月） |
| Lambda | 100万リクエスト/月（永続） |
| API Gateway | 100万API呼出/月（12ヶ月） |
| S3 | 5GB（12ヶ月） |
| CloudFront | 1TB転送/月（12ヶ月） |
| Cognito | 50,000 MAU（永続） |

**初年度はRDS無料枠により、パターンAで月額 ~$0-5 程度に抑えられる可能性がある。**

## 推奨

| 観点 | 推奨 |
|------|------|
| **プロトタイプ段階** | パターンB (Lightsail $5) — 最安、開発しやすい |
| **本番運用** | パターンA (サーバレス) — 保守最小化 |
| **外注前提** | パターンA or C — マネージドサービス活用で引き継ぎしやすい |
