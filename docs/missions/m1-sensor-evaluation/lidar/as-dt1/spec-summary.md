**更新日**: 2026-03-10

## 概要

AS-DT1はソニー製のLiDARデプスセンサー。2026年春発売予定。

---

## 主要仕様

### 測距性能

| 項目 | 値 |
|------|-----|
| 最大測距距離 | 20〜40m（モード依存） |
| 測距点数 | 288〜576点/フレーム |
| フレームレート | 最大30fps（33.33ms） |
| 屋内Accuracy | ±10〜±60mm |
| 屋外Accuracy | ±25〜±120mm（40000 lx下） |

### 物理仕様

| 項目 | 値 |
|------|-----|
| 外形寸法 | 29×29×31 mm（W×H×D） |
| 質量 | 約46g |
| 消費電力 | 最大2.5W |
| 動作温度 | -5〜+45℃ |

### インターフェース

| 項目 | 値 |
|------|-----|
| USB | USB 3.2 Gen1（Type-C） |
| UART | 115200〜921600 bps |
| 電源 | USB 5V または 外部12〜24V |
| トリガー | Trigger In/Out（5Vトレラント） |

### 耐久性

| 項目 | 値 |
|------|-----|
| 耐衝撃性 | 200G（7〜8 ms） |
| MTBF | 約10.9年 |

---

## 機能

### 測距モード

| モード | 機能 |
|--------|------|
| CDC | USB経由で測距結果を出力 |
| EXTUART | UART経由で測距結果を出力 |
| INTEN | 赤外線画像をUVC出力 |
| HIST | SPADセンサーの全データをUVC出力 |

### 出力形式

| 形式 | データ内容 | 備考 |
|------|----------|------|
| ascii | Z方向のみ | 低帯域向け（1.4Mbps） |
| binary | XYZ点群（1次・2次ピーク） | フル精度（2.1Mbps） |
| binz | Z方向のみ | 最低帯域（0.35Mbps） |

### 追加機能

- **障害物検出**: 指定距離範囲内の物体を検出、Trigger Out出力
- **IMU**: 加速度（±4G）・角速度（±500dps）、100Hz
- **時間同期**: タイムスタンプカウンター、外部トリガー

## SDK・開発環境

### 対応OS

- Windows 11 (64bit)
- Ubuntu 24.04
- Jetson Orin AGX (JetPack 6.2.1)
- Raspberry Pi OS

### 提供物

- サンプルアプリケーション（Windows exe + Pythonソース）
- Python API（点群処理、アップデート、デバイス検索）
- APIマニュアル（日本語/英語）

## 当社での使用実績

`SONY-LiDAR-TestMinutes`プロジェクトで動作確認済み:

- ROS2ドライバー: vendor提供（`as_dt1_ros2_driver`）
- 30M15Fモード: 屋外フィールドテストで15fps動作確認
- 座標系: X+=下、Y+=左、Z+=前（射出方向）
