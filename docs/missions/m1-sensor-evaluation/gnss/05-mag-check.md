## 1. 確認方法

GCS（QGroundControl等）の地図画面で、各機体の磁気方位（コンパスの向き）を目視比較。
全機体を同じ向きに置き、方位矢印のずれを確認する方式。

## 3. 発見事項

### No.02のMAG 45度ずれ

- シート内コメント: 「No.02が45度くらいずれている。キャリブレーションで解消できた 2/8」
- **2/8にキャリブレーションで解消済み** → 現在は問題なし
- 原因: 磁気干渉またはキャリブレーション不足

### No.05はMAGに関しては正常

- No.05のGNSS問題（L2受信不良・RTK FIX NG）は**磁気コンパスとは無関係**
- GNSS受信系の問題に限定される

### バッテリー状態

- 全機体67〜68%で均一 → バッテリー残量による性能差はない

## 4. 合格基準検討への示唆

| 項目 | 現状 | 方向性 |
|------|------|--------|
| MAGずれ量 | No.02が45度ずれ（キャリブで解消） | 許容ずれ角の基準設定が必要（例: ±5度以内） |
| 確認手順 | 目視（GCS画面で方位比較） | 定量的な角度測定が望ましい |
| キャリブ後の再確認 | 実施有無が不明確 | キャリブ後の確認を手順に含めるべき |
