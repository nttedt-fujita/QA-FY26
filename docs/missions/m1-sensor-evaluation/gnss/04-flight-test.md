## 4. 発見事項

### 全機体共通

- **ノイズ・ジャミングスパイクは全機体（Ref含む）で発生** → 環境要因（本社付近の電波環境）
- 衛星数→精度改善→RTK FIXの正常シーケンスはRef含め確認済み
- 革新比率は概ね0.5以下（良好）

### No.5固有の問題

| 項目 | No.5の状態 | 他機体 |
|------|-----------|--------|
| RTK FIX時間 | **5分以上 NG** | 5秒〜55秒 |
| 飛行中衛星数 | **19** | 25〜27 |
| 飛行中水平精度 | 0.899m | 0.474〜0.675m |
| 飛行中高さ精度 | **1.707m** | 0.481〜0.940m |
| ジャミングスパイク | 大きく頻度高い | 全機体であるが程度は軽い |

### No.2の不安定性

- RTK FIX後に**Floatに落ちやすい** → 受信環境が変化すると維持できない
- 2回目受信感度でG27, G10が「–」 → 受信感度にも弱点あり

### 他シートとの相関（No.5）

| データソース | No.5の異常 | 相関 |
|------------|-----------|------|
| 受信感度 | Q2 L1=13〜24dBHz, L2欠損多数 | L2受信不良 |
| スペアナ | RF2 PGA=38dB, 波形異常 | L2帯ハードウェア問題 |
| 飛行試験 | RTK FIX NG, 衛星19, ジャミング大 | L2不良→RTK不可 |

→ **L2帯の受信不良がRTK FIX不可の根本原因**。RTK FIXにはL1+L2の二周波が必要。
