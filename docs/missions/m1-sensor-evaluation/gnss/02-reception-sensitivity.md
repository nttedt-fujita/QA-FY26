## 5. 発見事項

### No.5の受信感度異常

- 全3回の測定で一貫してQZSSのL1が低い（24→43→**13** dBHz）
- GPS衛星のL2受信が他機体より欠損が多い（「–」が多い）
- u-centerの棒グラフでも視覚的に低い

### QZSS L2は全機体で受信不可

- 仰角テーブルでもQ2/Q3/Q7全てL2が「–」
- Ref機でも同様 → 機体固有の問題ではなく**設定または仕様**
- F9Pの設定かアンテナ特性の問題か要確認

### Ref機と実機の差

- Refは全衛星で最も高いC/N0
- 実機（No.1〜5）はRefより**5〜15dBHz低い**のが一般的
- バッテリー搭載・アンテナ取付環境の影響か、モジュール個体差かは不明

### 「0922」等のシルクなし機体

- 受信レベルはRefに近い → 正常機体
- No.1〜5（小峰GPS搭載機）との比較用に測定された
