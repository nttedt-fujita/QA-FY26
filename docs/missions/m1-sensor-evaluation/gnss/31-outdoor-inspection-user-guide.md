# 屋外検査 ユーザーガイド

---
作成: Session 131 (2026-03-12)
出典: prototype/m1-gnss 実装コード
---

## 概要

GNSS受信機の屋外検査を実行し、以下の4項目を評価する。

| 項目 | 合格基準 | 根拠 |
|------|---------|------|
| L1最小C/N0 | ≥30 dBHz | ADR-008 |
| L2受信率 | ≥50% | ADR-008 |
| RTK FIX時間 | ≤30秒 | ADR-008 |
| RTK FIX率 | >95% | ADR-008 |

---

## 処理フロー

```
┌─────────────────────────────────────────────────────────────┐
│ 1. 装置接続（/devices画面）                                   │
│    - シリアルポートを選択して接続                              │
│    - ステータスが「接続中」になることを確認                     │
└─────────────────────────────────────────────────────────────┘
                              ↓
┌─────────────────────────────────────────────────────────────┐
│ 2. 屋外検査画面へ移動（/inspections/outdoor）                 │
└─────────────────────────────────────────────────────────────┘
                              ↓
┌─────────────────────────────────────────────────────────────┐
│ 3. ロット選択（任意）                                        │
│    - 検査対象のロットを選択                                   │
│    - 未選択でも検査は実行可能                                  │
└─────────────────────────────────────────────────────────────┘
                              ↓
┌─────────────────────────────────────────────────────────────┐
│ 4. 検査時間選択                                              │
│    - 30秒 / 60秒 / 120秒 / 5分 から選択                       │
│    - ADR-008では30秒以上を推奨                                │
└─────────────────────────────────────────────────────────────┘
                              ↓
┌─────────────────────────────────────────────────────────────┐
│ 5. 「検査開始」ボタン押下                                     │
│    - 装置が接続されていないと押せない                          │
└─────────────────────────────────────────────────────────────┘
                              ↓
┌─────────────────────────────────────────────────────────────┐
│ 6. 検査中（サンプル収集）                                     │
│    - 各パネルがAPIポーリングでデータ取得                       │
│    - NAV-STATUS: Fix状態（carrSoln）                         │
│    - NAV-SIG: 衛星信号（C/N0、L2有無）                        │
│    - MON-SPAN: スペクトラム（表示のみ）                        │
│    - スカイプロット: 衛星配置（表示のみ）                      │
│                                                              │
│    画面表示:                                                  │
│    - 残り時間（カウントダウン）                                │
│    - サンプル数                                               │
└─────────────────────────────────────────────────────────────┘
                              ↓
┌─────────────────────────────────────────────────────────────┐
│ 7. 検査終了                                                  │
│    - 時間終了で自動終了                                       │
│    - または「検査停止」ボタンで手動終了                         │
└─────────────────────────────────────────────────────────────┘
                              ↓
┌─────────────────────────────────────────────────────────────┐
│ 8. 検査結果表示                                              │
│    - 合格/不合格の判定                                        │
│    - 各項目の値と合否                                         │
│    - 不合格の場合は理由を表示                                  │
└─────────────────────────────────────────────────────────────┘
                              ↓
┌─────────────────────────────────────────────────────────────┐
│ 9. 「結果を保存」ボタン押下 ★重要★                            │
│    - このボタンを押さないとDBに保存されない                     │
│    - 保存成功で「保存済み (ID: XX)」と表示                     │
│    - 保存後は「新規検査」で次の検査へ                          │
└─────────────────────────────────────────────────────────────┘
```

## 状態遷移

```
       ┌─────────────┐
       │    idle     │ ← 初期状態
       │  （待機中）  │
       └──────┬──────┘
              │ 検査開始
              ↓
       ┌─────────────┐
       │   running   │ ← サンプル収集中
       │  （検査中）  │
       └──────┬──────┘
              │ 時間終了 or 検査停止
              ↓
       ┌─────────────┐
       │  completed  │ ← 結果表示・保存可能
       │  （完了）    │
       └──────┬──────┘
              │ 新規検査
              ↓
       ┌─────────────┐
       │    idle     │ ← リセット
       └─────────────┘
```

## よくあるエラー

### 「検査開始」ボタンが押せない

- **原因**: 装置が接続されていない
- **対処**: /devices画面で装置を接続する

### 検査完了したがサンプル数が0

- **原因**: バックエンドが起動していない、またはAPIエラー
- **対処**: バックエンドのログを確認

### 全項目不合格

- **原因**: 屋内で検査した（RTK FIXしない、衛星が見えない）
- **対処**: 屋外の空が見える場所で検査する

### 「結果を保存」でエラー

- **原因**: バックエンドが起動していない、またはDB接続エラー
- **対処**: バックエンドのログを確認

## 技術詳細

### データフロー

```
                   ポーリング
┌─────────┐   GET /api/xxx   ┌─────────┐
│ フロント │ ───────────────→ │ バック  │
│ エンド  │ ←─────────────── │ エンド  │
└────┬────┘   レスポンス     └────┬────┘
     │                            │
     │ onSample()                 │ UBXパース
     ↓                            ↓
┌─────────┐                  ┌─────────┐
│  Hook   │                  │  GNSS   │
│ 状態管理 │                  │  受信機  │
└────┬────┘                  └─────────┘
     │
     │ 集計処理（FE側）
     ↓
┌─────────┐   POST /api/...   ┌─────────┐
│ 検査結果 │ ───────────────→ │ DB保存  │
└─────────┘                  └─────────┘
```

### 集計処理の実行場所

- **サンプル蓄積**: フロントエンド（React状態）
- **集計計算**: フロントエンド（検査完了時）
- **DB保存**: バックエンド（API呼び出し時）
