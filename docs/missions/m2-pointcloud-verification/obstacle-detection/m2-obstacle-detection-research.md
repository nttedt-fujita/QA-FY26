# M2: 障害物検知評価アプローチの調査

**作成日**: 2026-03-09 (Session 52)
**背景**: M2「点群データ検証」の実態は「障害物検知システムの評価」に近い

---

## 調査の結論

**AirGrowの用途（リアルタイム障害物検知）に対しては、測量用の点群精度評価（ASPRS基準等）ではなく、障害物検知性能の評価が必要**

---

## 評価指標（藤田さん案 + 調査結果）

### 1. 検知距離

| 指標 | 定義 | 参考値 |
|------|------|--------|
| **検知距離** | 障害物を検知できる最大距離 | Vision系: 15-30m |
| **最小安全距離** | 衝突回避に必要な最小距離 | 4-6m（中型ドローン） |

### 2. 反応時間

| 指標 | 定義 | 参考値 |
|------|------|--------|
| **反応時間** | 検知から回避動作開始までの遅延 | Vision系: 50-200ms |
| **システムレイテンシ** | センサ入力から制御出力までの全体遅延 | 高速システム: 3.5ms |

### 3. 検知精度

| 指標 | 定義 | 参考値 |
|------|------|--------|
| **検知精度** | 障害物を正しく検知する割合 | 深層学習系: 98% |
| **False Alarm率** | 存在しない障害物を検知する割合 | ← **AirGrowの課題** |
| **見逃し率** | 存在する障害物を見逃す割合 | - |

### 4. 速度依存性

| 飛行速度 | 必要検知距離 | 備考 |
|---------|------------|------|
| 4 m/s | 7m以上 | 反応時間350ms想定 |
| 15 m/s | 6m（超音波）で0.4s | ギリギリ |

## 業界の評価基準・規格

### EUROCAE ED-267（欧州規格）

- **正式名称**: OSED for Detect and Avoid in Very Low Level Operations
- **発行**: 2020年、EUROCAE WG-105 SG-13
- **内容**: 低高度UAS運用におけるDAA（Detect and Avoid）の運用要件定義
- **対象**: 交通、固定/移動障害物、地形、気象、野生動物、地上の人

**位置づけ**:
- MASPS（最小航空システム性能標準）
- MOPS（最小運用性能仕様）の基礎

**参照**: [EUROCAE ED-267](https://www.eurocae.net/ed-267-osed-for-detect-and-avoid-in-very-low-level-operations/)

### FAA Part 108（米国、策定中）

- BVLOS（目視外飛行）向けのDAA要件
- 2025年後半に追加規定（Part 74）リリース予定

## 商用ドローンの実装例

### Skydio 2

| 項目 | 仕様 |
|------|------|
| センサ | 6台の4Kカメラ（360°全方位） |
| プロセッサ | NVIDIA Tegra X2 |
| 回避可能速度 | 最大36 mph（約16 m/s） |
| 更新レート | 100万点/秒以上 |
| 処理方式 | 9つのカスタムDNNで3Dモデル構築 |

**注意**: 公式の障害物回避仕様値は「推定値」であり、環境やフライトモードにより変動

**参照**: [Skydio障害物回避説明](https://dronedj.com/2019/10/01/skydios-obstacle-avoidance-self-flying-capability/)

### DJI

- 複数センサ（Vision + ToF + 超音波）の組み合わせ
- 機種ごとに検知範囲・対応角度が異なる

**参照**: [DJI障害物回避システム](https://support.dji.com/help/content?customId=en-us03400006547&spaceId=34&re=US&lang=en&documentType=artical&paperDocType=paper)

## 評価試験方法（案）

### 静的試験

| 試験 | 内容 | 評価指標 |
|------|------|---------|
| 検知距離試験 | 既知距離に障害物を配置、検知可否を確認 | 最大検知距離 |
| 角度依存性試験 | 障害物の角度を変えて検知率を測定 | 角度別検知率 |
| 素材依存性試験 | 反射率の異なる素材での検知率 | 素材別検知率 |

### 動的試験

| 試験 | 内容 | 評価指標 |
|------|------|---------|
| 飛行速度別試験 | 速度を変えて回避成功率を測定 | 速度別回避成功率 |
| 反応時間測定 | 障害物出現から回避開始までの時間 | 平均反応時間 |
| False Alarm試験 | 障害物なし環境での誤検知回数 | False Alarm率 |

### フィールド試験

| 試験 | 内容 | 評価指標 |
|------|------|---------|
| みかん畑模擬環境 | 実運用に近い環境での飛行試験 | 衝突回数、誤検知回数 |
| 境界条件試験 | 細い枝、ワイヤーなど検知困難な障害物 | 見逃し率 |

## AirGrow向け評価項目（案）

### 優先度高

| 項目 | 理由 |
|------|------|
| **False Alarm率** | 現在の課題、最優先で評価 |
| **検知距離** | 回避動作に必要な余裕を確保できるか |
| **反応時間** | 飛行速度との関係で安全性に直結 |

### 優先度中

| 項目 | 理由 |
|------|------|
| 見逃し率 | 衝突リスクに直結 |
| 速度依存性 | 運用速度での安全性確認 |

### 優先度低（将来）

| 項目 | 理由 |
|------|------|
| 環境条件依存性 | 日照、天候による影響 |
| 長期安定性 | センサ劣化による性能変化 |
