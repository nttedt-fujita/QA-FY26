## 2. 問題点の整理

### M1が重すぎる問題

**M1 = Lidar + GNSS の2アイテムを抱えている**

| サブミッション | 状況 | 負荷 |
|---------------|------|------|
| M1-A: Lidar | ゼロから。規格調査・環境構築が必要 | **高** |
| M1-B: GNSS | 既存知見あり。ヒアリングで把握可能 | **中** |

→ **M1を1つのミッションとして扱うのは無理がある可能性**

### 調査が全くできていない問題

Session 1-2で「規格名」「指標名」は調べたが：
- **自社でどう適用するか**が全く具体化されていない
- **評価環境がない**ことすら把握できていなかった
- **小板橋さんの知見**を聞いていなかった

→ **1次調査（ヒアリング + Web調査）が必須**
