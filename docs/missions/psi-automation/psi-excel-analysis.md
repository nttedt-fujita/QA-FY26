## 1. シート分類

### A. PSI本体（週次スナップショット）— 40シート

| シート名パターン | 例 | 件数 | サイズ |
|-----------------|-----|------|--------|
| PSI_{日付} | PSI_Mar23, PSI_Mar16, ... | 40枚 | 各374行×62列（最新）〜198行×56列（古い） |

- **FY25の4月〜3月の各週**のスナップショットを1シートずつ保持
- 製品×行項目（製造/販売/在庫等）×月のマトリクス
- **行が増えている** = 新製品が追加されている（198行→374行）
- 列構成: FY23〜FY26の月別（4月〜3月）

### B. 受注案件一覧_Agri — 1シート

| 項目 | 内容 |
|------|------|
| 行数 | 694行（ヘッダ含む） |
| 主要カラム | Kintone#, 商流(N系/S系/I系/D系), 受注日, 顧客, 納品希望日, 出荷月, 機体数, バッテリ数, チャージャー数, 粒剤数 |
| 識別子 | NSALES-XXXXX（左端）+ Kintone#（2列目） |

**→ kintone出力データの可能性が非常に高い**（Kintone#カラムあり）

## 2. PSI本体シートの行項目構造（PSI_Mar23）

製品ごとに以下の行項目がセットで並ぶ:

| 行項目 | 説明 | データソース推定 | 確度 |
|--------|------|-----------------|------|
| 製造 | 製造数（ナカヨ出荷ベース等） | 不明 | **D** — 「ナカヨ」の記載はあるがGEN等のシステム由来かは未確認 |
| ┗製造（ナカヨ出荷ベース） | 製造の内訳 | 不明 | **D** |
| ┗製造（生産投入ベース） | 製造の内訳 | 不明 | **D** |
| ┗入庫戻し | 返品等 | 不明 | **D** |
| 販売（受注） | 受注数 | kintoneの情報由来 | **B** — AgriにKintone#あり(A) + S313ヒアリング |
| 販売（NTT） | NTT商流の販売 | kintoneの情報由来 | **D** — AC101旧モデルにしかない行。推測 |
| 販売（卸SL） | SL商流の販売 | kintoneの情報由来 | **D** — 同上 |
| 納品（売上） | 納品実績 | 不明 | **D** — GEN由来の可能性あるが未確認 |
| 出荷予測 | 出荷の予測値 | 予測シート（長屋さん） | **C** — S313ヒアリングのみ |
| 受注数（出荷ベース） | 受注数の別集計 | Agriシートの集計 | **A** — 突合で数値一致を確認済み |
| 出庫実績 | 実際の出庫 | 不明 | **D** — 群馬通商出庫数との直接一致は未確認 |
| ┗納品 | 出庫の内訳 | 不明 | **D** |
| ┗リコール出荷 | 出庫の内訳 | 不明 | **D** |
| ┗仕損品 | 出庫の内訳 | 不明 | **D** |
| ┗社内用途 | 出庫の内訳 | 不明 | **D** |
| 在庫（受注ベース残） | 在庫計算値 | 自動計算 | **A** — 凡例「青塗りつぶし = 自動計算」で確認 |
| 在庫（納品ベース残） | 在庫計算値 | 自動計算 | **A** |
| 在庫（出荷予測ベース残） | 在庫計算値 | 自動計算 | **A** |
| 在庫（受注数残） | 在庫計算値 | 自動計算 | **A** |
| 在庫(NTT) / 在庫(SL) / 在庫(TTL) | 商流別在庫 | 不明 | **D** — PSIコメントに「群馬倉庫」記載あり(A)だが数値突合は未実施 |

### PSI本体の凡例（シート上部の注記）

- ★青塗りつぶし = **自動計算**
- ★ピンク = **滝澤さん出荷予測（経営ベース）**
- ★黄色 = **注文書受領数**

## 3. データソース推定マップ

```
[kintoneの情報]                                        ... [B] AgriにKintone#あり + S313ヒアリング
  ├→ 受注案件一覧_Agriシート                           ... [B] 転記方法は未確認（CSV? 画面から手入力?）
  └→ PSI本体の「受注数（出荷ベース）」行              ... [A] 突合で数値一致を確認済み

[GEN?]                                                 ... [D] GENがソースかは未確認
  ├→ PSI本体の「製造」行?                              ... [D]
  └→ PSI本体の「納品（売上）」行?                      ... [D]

[群馬通商Excel]                                        ... [D] PSIとの数値突合は未実施
  ├→ PSI本体の「出庫実績」行?                          ... [D] 出庫数の直接一致は未確認
  └→ PSI本体の「在庫(NTT)」「在庫(SL)」行?            ... [D] コメントに「群馬倉庫」記載あり[A]

[長屋さん（営業支援）]                                 ... [C] S313ヒアリングのみ
  └→ 予測シート → PSI本体の「出荷予測」行

[滝澤さん]                                            ... [A] PSI凡例「ピンクは滝澤さん出荷予測」
  └→ PSI本体の「出荷予測（経営ベース）」

[自動計算（Excel数式）]                                ... [A] 凡例「青塗りつぶし = 自動計算」
  └→ 在庫（受注ベース残）、在庫（納品ベース残）等
```
